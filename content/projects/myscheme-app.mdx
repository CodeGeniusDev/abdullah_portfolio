**My Scheme** is a modern and community-driven platform that allows users to:
- Explore latest government schemes in an organized format
- Submit new schemes with images, categories, and rich descriptions
- Manage submitted schemes through an intuitive dashboard
- Authenticate securely using NextAuth with MongoDB
- Enjoy a clean, responsive UI built with Tailwind CSS and Next.js App Router
